powerROC v0.1

For the evaluation of clinical prediction models, the TRIPOD+AI statement mandates reporting the process of determining the sample size and justifying its sufficiency in addressing the research question.

powerROC is a web application that helps researchers determine the sample size required to estimate the area under the receiver operating characteristic curve (AUROC) with a desired level of precision or to compare the AUROCs of two models with a desired level of power.

  • For evaluating a single prediction model

    • click Sample size to precisely estimate the AUROC.

  • For comparing two prediction models

    - With acess to a pilot test set (or a validation set from which you may want to vary the prevalence),

    click Sample size for comparing AUROCs: using a pilot test set

    to determine sample size based on pilot data.

    - Without acess to a pilot test set or a validation set,

    click Sample size for comparing AUROCs: specifying joint distributions

    to calculate sample size without needing preliminary data.
